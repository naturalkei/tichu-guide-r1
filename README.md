# tichu-guide-r1

티츄 보드게임 가이드

---

## 프로젝트 취지

* 티츄의 규칙서는 인터넷에 쉽게 찾아 볼 수 있지만, 광고없이 모바일(핸드폰) 화면에서 깔끔하게 게임 방법 및 규칙을 확인 할 수 있는 잘 만들어진 (한국어) 가이드 사이트가 없어서 이 프로젝트를 만듬

---
## OVERVIEW

* 보드게임 티츄 가이드 오픈소스 프로젝트
* 티츄 게임의 준비 방법
* 티츄 게임의 규칙 설명
* 점수계산 방식 (점수 계산 팁)
* 게임 진행간 주의 및 에티켓 도움말
* 모바일 우선 UI 디자인
* 티츄 게임 가이드 오픈소스 프로젝트의 라이센스 및 disclaimer 도 명확히 작성하라
* conventional commit 규칙에 따라 커밋 메시지를 작성
* 모든 지시사항을 개별 브랜치로 생성하여 작업하고 main에 병합하는 형태로 작업할것
